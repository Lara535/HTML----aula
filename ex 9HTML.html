Wb Semântica

A web semnântica é uma extensão da web atual, onde a informação possui significado que permite que computadores e pessoa possam trabalhar em conjunto. As tags da web semântica ajudam os mecanismos de busca de tarefa de criar repositórios e recuperar informações, em que as principais tags são: <header>, <nav>, <main>, <section>, <article>, <aside> e <footer>.

Neste exemplo estamos usando a web semântica em todo o documento html.
